# Design System

## 컬러 팔레트

### Primary Colors
- Primary: `#1976d2`
- Secondary: `#dc004e`
- Background: `#ffffff`
- Surface: `#f5f5f5`

## 타이포그래피

- 기본 폰트: Roboto, Helvetica, Arial, sans-serif
- H1: 2.125rem / font-weight: 500
- H2: 1.875rem / font-weight: 500
- Body: 1rem / font-weight: 400

## 간격 (Spacing)
- 기본 단위: 8px (MUI spacing(1) = 8px)

## 컴포넌트 스타일 가이드
- 버튼: variant="contained" 기본 사용
- 카드: elevation={2} 기본 사용
- 입력 필드: variant="outlined" 기본 사용
